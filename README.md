# toDoApp

I created this toDoApp using html,css,js. Totally from scratch and with the help of few courses and stackoverflow.

You can create to do list items, you can check them and delete them.
- state will be preserved in localstorage
- list items can be entered with help of enter button
- list items with same content can exist
- an id is assigned the moment an element is created
- some of the logic is divided into functions
